# 💃 Salsa Romántica FM

Una experiencia de streaming premium para los amantes de la salsa. Este proyecto es una aplicación web moderna diseñada para ofrecer una transmisión de audio ininterrumpida con una interfaz visualmente impactante.

## ✨ Características Principales

* **Persistencia de Audio:** Gracias a una arquitectura de *Layout Global*, la música nunca se detiene mientras navegas por las diferentes secciones del sitio.
* **Metadatos en Tiempo Real:** Integración con la API de ZenoFM mediante *Server-Sent Events* (SSE) para mostrar el artista y la canción actual al instante.
* **Interfaz Neón:** Diseño moderno utilizando Tailwind CSS, con efectos de brillo y una estética oscura optimizada para la experiencia nocturna.
* **Control de Volumen Profesional:** Slider de alta precisión construido sobre Radix UI para un manejo suave y accesible.

## 🛠️ Stack Tecnológico

* **Frontend:** React.js con Vite.
* **Estilos:** Tailwind CSS + Lucide Icons.
* **Componentes UI:** Radix UI (Primitives).
* **Enrutamiento:** React Router Dom (v6).
* **Streaming:** ZenoFM Metadata API (SSE).

## 🚀 Instalación y Configuración

1. **Clonar el repositorio:**
   ```bash
   git clone git@github.com:Jzea94/La_Romantica_Radio.git
   cd salsa-romantica-fm
   ```

2. **Instalar dependencias**
   ```bash
   npm install
   ```

3. **Ejecutar en desarrollo**
   ```bash
   npm run dev
   ```
